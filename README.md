# Atividades-com-Strings-e-de-Logica-em-JavaScript
Este repositório contém uma coleção de 23 exercícios práticos focados na manipulação de variáveis, strings e lógica de programação em JavaScript. O projeto demonstra o uso de métodos essenciais para o tratamento de dados, desde buscas simples e formatação de texto até o uso de expressões regulares e funções de processamento de arrays.
